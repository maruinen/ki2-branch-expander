# プロジェクト概要: Ki2 Branch Expander
同一局面を認識しないKI2再生アプリ向けに、合流した局面からの分岐をすべての経路に複製・展開するツール。

## ターゲット
- 同一局面を自動合流（グラフ表示）できないKI2ビューアを利用しているユーザー。

## 主要機能
1. **KI2パース**: 分岐を含むKI2ファイルを読み込み、内部で指し手ツリーを構築する。
2. **局面同一性判定**: 
   - `python-shogi` を用いて、盤面（駒配置、手番、持駒）をシミュレートする。
   - 異なる手順でも、同一の盤面状態になった場合は「同一局面」とみなす。
3. **分岐の全展開**:
   - ある局面 $S$ において指された手 $\{M_1, M_2, ...\}$ を、その局面 $S$ に至るすべての経路の末尾に結合する。
4. **KI2出力**: 展開されたツリー構造を再度KI2形式のテキストファイルとして書き出す。

## 技術スタック
- 言語: Python 3.10+
- ライブラリ: python-shogi
